# Markdown样式增强

## 🎨 优化内容

### 1. 标题层级优化

#### 修复前
```css
/* 原始样式 - 层级不明显 */
h1, h2 { font-size: 15px, 14px } /* 差异很小 */
h3-h6 { font-size: 13px, 12px }
```

#### 修复后
```css
/* 优化样式 - 层级分明 */
h1 { font-size: 20px; color: #1a202c; font-weight: bold; }
h2 { font-size: 18px; color: #2d3748; font-weight: bold; }
h3 { font-size: 16px; color: #4a5568; font-weight: bold; }
h4 { font-size: 15px; color: #718096; font-weight: 600; }
h5 { font-size: 14px; color: #a0aec0; font-weight: 600; }
h6 { font-size: 13px; color: #cbd5e0; font-weight: 500; }
```

### 2. 背景透明化

#### 代码块背景
```css
/* 修复前 */
background-color: #f5f5f5;  /* 不透明白色 */
border: 1px solid #e1e4e8;

/* 修复后 */
background-color: rgba(240, 240, 240, 0.3);  /* 半透明 */
border: 1px solid rgba(225, 228, 232, 0.5);
```

#### 行内代码背景
```css
/* 修复前 */
background-color: #f5f5f5;  /* 不透明白色 */

/* 修复后 */
background-color: rgba(240, 240, 240, 0.2);  /* 更透明 */
border: 1px solid rgba(225, 228, 232, 0.3);
```

### 3. 新增样式支持

#### 引用块
```css
blockquote {
  border-left: 4px solid rgba(66, 153, 225, 0.6);
  padding-left: 12px;
  background-color: rgba(237, 242, 247, 0.3);
  font-style: italic;
  color: #4a5568;
}
```

#### 强调文本
```css
strong {
  font-weight: bold;
  color: #2d3748;
}

em {
  font-style: italic;
  color: #4a5568;
}
```

## ✨ 支持的完整Markdown格式

### 📝 文本格式
- **粗体文本** - 深色强调
- *斜体文本* - 中等颜色
- ~~删除线文本~~
- `行内代码` - 透明背景框

### 📋 结构格式
- # 一级标题 (20px, 最突出)
- ## 二级标题 (18px, 很突出)  
- ### 三级标题 (16px, 突出)
- #### 四级标题 (15px, 中等)
- ##### 五级标题 (14px, 较小)
- ###### 六级标题 (13px, 最小)

### 💬 块级元素
- > 引用块 - 带左边框和透明背景
- 代码块 - 透明背景，保持换行
- 列表项 - 优化间距

### 🔗 链接
- [链接文本](URL) - 保持原始样式

## 🛠️ 实现细节

### 透明度设计原则
- **代码块**: `rgba(240, 240, 240, 0.3)` - 30%透明度
- **行内代码**: `rgba(240, 240, 240, 0.2)` - 20%透明度  
- **引用块**: `rgba(237, 242, 247, 0.3)` - 30%透明度
- **边框**: `rgba(225, 228, 232, 0.5)` - 50%透明度

### 间距设计
- **标题**: 上下margin根据层级递减
- **代码块**: 上下8px间距
- **列表项**: 下方4px间距
- **引用块**: 上下8px间距
